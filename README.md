# Michael Cantor - Artist Link Page

A minimal, mobile-friendly link-in-bio site built with Next.js and Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

## Add Your Images

Place these images in the project:

- `public/profile.jpg` — Profile photo (square, at least 224x224px)
- `public/releases/core-memories.jpg` — Core Memories cover art
- `public/releases/lips-of-an-angel.jpg` — Lips of an Angel cover art
- `public/releases/change-my-name.jpg` — Change My Name cover art

## Deploy to Vercel

1. Push this repo to GitHub
2. Import the project on [vercel.com](https://vercel.com)
3. Deploy — no configuration needed

## Customize

Edit `src/app/page.tsx` to update links, bio text, or add new sections.
